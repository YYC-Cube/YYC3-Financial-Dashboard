## 测试环境

### 配置（vitest.config.ts）
- **环境**：jsdom（模拟浏览器）
- **globals**：true（无需导入 describe/it/expect）
- **路径别名**：`@/` → 项目根目录

### 测试文件位置
- `__tests__/` 目录
- 命名：`*.test.ts` 或 `*.test.tsx`

### CI 集成
- GitHub Actions 中运行 `pnpm test`
- 测试失败阻止部署
