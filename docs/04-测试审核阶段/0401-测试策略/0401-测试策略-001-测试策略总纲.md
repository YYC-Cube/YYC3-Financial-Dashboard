## 测试策略

### 测试框架
- **运行器**：Vitest 4.x
- **环境**：jsdom
- **React 测试**：@testing-library/react

### 测试分层
1. **单元测试**：i18n 引擎、工具函数
2. **组件测试**：UI 组件渲染和交互
3. **集成测试**：页面级功能验证
4. **E2E 测试**：（规划中）Playwright

### 当前覆盖
- `__tests__/i18n/engine.test.ts`：12 个测试（翻译/插值/切换/缓存/订阅/回退/统计）

### 运行命令
```bash
pnpm test          # 运行一次
pnpm test:watch    # 监听模式
```
