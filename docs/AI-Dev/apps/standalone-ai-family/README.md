# standalone-ai-family — AI Family 独立版 (中枢)

## 启动

```bash
pnpm --filter @yyc3/app-standalone-ai-family dev
```

## 效果

```
欢迎弹窗 → AI Family 首页 + 中枢浮窗 (绿色 #00FF88)
浮窗: 对话 / 家人命令 / 人格条 / 提示词 / 配置
```

## 特色

- 8 位家人人格条 (千行/万物/先知/伯乐/天枢/守护/宗师/灵韵)
- 8 条家人呼叫命令
- 独立主题色 `#00FF88`

## 浮窗命令

- 呼叫千行 / 万物 / 先知 / 伯乐 / 天枢 / 守护 / 宗师 / 灵韵
