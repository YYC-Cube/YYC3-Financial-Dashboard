## 详细内容

## 应用表现层

### 页面结构
- `/`：首页（重定向到仪表盘）
- `/dashboard`：仪表盘主页面

### 组件树
```
RootLayout
├── ThemeProvider（明暗主题）
├── I18nApp（多语言 Provider）
│   ├── Layout
│   │   ├── Sidebar（侧边栏，md+ 显示）
│   │   ├── TopNav（顶部导航）
│   │   │   ├── ThemeToggle
│   │   │   ├── LanguageSwitcher
│   │   │   └── Profile
│   │   └── DashboardContent
│   │       ├── SpendingTrendChart（动态导入）
│   │       ├── ExpenseCategoryChart（动态导入）
│   │       ├── List01（账户列表）
│   │       ├── List02（交易列表）
│   │       └── List03（目标列表）
│   ├── PWARegister
│   └── PWAInstallPrompt
```
