# packages — 插件包目录

每个包可以独立 `pnpm build`，也可被 `apps/standalone-*` 引用。

| 包 | 说明 |
|----|------|
| `shell/` | 系统外壳 (事件总线/存储/Hub 浮窗/欢迎页) |
| `plugin-monitor/` | 监控中心 |
| `plugin-ops/` | 运维管理 |
| `plugin-ai/` | AI 智能 |
| `plugin-ai-family/` | AI Family (中枢) |
| `plugin-dev/` | 开发工具 |
| `plugin-admin/` | 系统管理 |

每个包内的 `README.md` 有详细说明。
