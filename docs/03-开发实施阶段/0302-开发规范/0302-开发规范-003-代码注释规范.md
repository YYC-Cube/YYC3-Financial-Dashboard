## 注释规范

### 文件头注释
每个 `.ts/.tsx` 文件必须有文件头：
```typescript
/**
 * @file 文件名称
 * @description 文件描述
 * @author YYC³
 * @version 1.0.0
 * @created 2026-07-15
 */
```

### 函数注释
```typescript
/**
 * 翻译核心方法
 * @param key - 翻译键（点分隔）
 * @param params - ICU 参数
 * @returns 翻译后的字符串
 */
```

### 行内注释
- 仅在逻辑不直观时添加
- 使用中文注释
- 避免显而易见的注释
